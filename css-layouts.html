CSS Grid

CSS Grid Layout er den nyeste og definitivt beste måten å lage et sideoppsett. Med Grid kan man enkelt kontrollere både rader og kolonner, og velge nøyaktig hvor de forskjellige elementene skal plasseres.

Grid kan settes opp på følgende måte; 

.grid {
    display: grid;
    grid-template-columns: 3fr 1fr;
    grid-gap: 1em;
    grid-auto-columns: min-content;     
    padding:1em;
    max-width: 95vw;
}

"display: grid;" gir tilgang til grid-funksjonalitet
"grid-template-columns:" velger hvordan kolonnene skal deles opp
"grid-gap:" spesifiserer mellomrom mellom elementene 

.header {
    background: url(headerBackground.png);
    background-size: 100% 100%;
    padding:0.4em;
    grid-column: 1/3;
    grid-row: 1/3;                       
    border-radius: 5px;
    color: #5e6e6b;
    text-shadow: 0.3em 0.3em 0.3em #000000;
    font-size: 1.2em;
    text-align: center;
}

"grid-column:" og "grid-row:" spesifiserer hvor i grid elementet skal være. 

.grid aside {
    display: grid;
    grid-auto-rows: minmax(100px, auto);
    padding: 1em;
    background:rgba(255, 255, 255, 0.1);
    color: #bbbbbb;
    padding:1em;
    border-radius: 5px;

}

Her setter vi en grid inne i en grid. 

.grid footer {
    background:rgba(255, 255, 255, 0.1);
    color: #bbbbbb;
    padding:1em;
    grid-column: 1/3;
    border-radius: 5px;
    text-align: center;

    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-gap: 1em;
    grid-auto-columns: min-content;
    padding:1em;
}

Her setter enda en grid inne i en grid. 

.grid main {
    background:rgba(255, 255, 255, 0.1);
    color: #bbbbbb;
    padding:1em;
    grid-column: 1/2;
    border-radius: 5px;
}

Her spesifiserer vi at elementet ".grid main" kun skal strekke seg over en kolonne. "grid-column: 1/2;" kan vi oversette til "strekk deg fra den første veggen i den første kolonnen, til den første veggen i den andre kolonnen".

CSS Flexbox

CSS Flexbox er en annen måte å lage sideoppsett. Når man skal bruke Flexbox, er det alltid viktig å tenke på hvilken retning elementene sorteres. Det kan vi bestemme ved å bruke flex-direction. Flex-direction har fire mulige values;

  • row
  • row-reverse
  • column
  • column-reverse

To viktige egenskaper er align-items og justify-content. Egenskapen "align-items" justerer elementene på tverraksen, og "justify-content" brukes til å justere elementene på hovedaksen.
Mulige values for align-items er;

  • stretch
  • flex-start
  • flex-end
  • center

Mulige values for justify-content er;

  • flex-start
  • flex-end
  • center
  • space-around
  • space-between
  • space-evenly
